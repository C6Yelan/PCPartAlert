# PCPartAlert 專案規劃

## 1. 專案目標
本專案旨在建立一個以 Discord Bot 為主的硬體價格追蹤和提醒工具，協助台灣使用者追蹤原價屋（CoolPC）電腦零件的零售價格變動，並在符合條件時收到通知。透過自動化的價格追蹤和通知機制，減少使用者手動查價的負擔，並讓他們能更快地掌握關注商品的價格變化。網站則作為輔助工具，提供更方便的通知條件設定介面，但不以完整商品查詢或比價瀏覽為主要功能。

## 2. 專案定位
PCPartAlert 定位為一個專注於價格的提醒工具，核心功能為追蹤、條件判斷和通知。目前以 Discord Bot 作為主要的互動和通知平台，網站則提供輔助性的設定功能。專案不涉及完整的商品查詢、規格整理、比價瀏覽、組單估價等功能，也不追蹤其他零售平台或官方建議售價。專注於提供一個簡單、有效的價格追蹤和通知系統，讓使用者能快速設定條件並接收通知。

## 3. 目標使用者
- 想追蹤特定零件價格的台灣使用者
- 不想每天手動查價，但又想掌握價格變動的使用者
- 需要針對特定商品設定追蹤條件並接收通知
- 以 Discord 為主要溝通平台的使用者

## 4. 想解決的問題
1. 手動查詢原價屋零件價格麻煩且耗時。
2. 使用者通常只關注少數幾個商品，沒必要反覆手動查詢全部商品。
3. 零售價格可能在短時間內更動，使用者難以及時發現。
4. 缺少一個可自訂條件並主動通知的簡單工具。

## 5. 核心需求
1. 使用者可以指定要追蹤的商品（如特定型號的顯示卡）。
2. 使用者可以設定價格變動的條件（如低於特定價格、短時間內降價幅度等）。
3. 系統會定期追蹤原價屋的零件零售價格，並判斷是否符合使用者設定的條件。
4. 當價格變動符合條件時，透過 Discord Bot 發送通知給使用者。
5. 提供一個網站介面，讓使用者更方便地設定追蹤條件，但不以完整商品查詢或比價瀏覽為主要功能。

## 6. 明確不做
- 不做完整商品查詢、瀏覽介面。
- 不做硬體規格整理和比較功能。
- 不做組單和估價功能。
- 不追蹤其他零售平台的價格。
- 不追蹤官方網站的價格或建議售價。
- 不以網站推播為主的通知系統。
- 不做市場分析或趨勢研究型內容。

## 7. 第一版範圍
- 追蹤原價屋（CoolPC）的電腦零件零售價格。
- 提供使用者設定追蹤商品與通知條件的方式。
- 判斷價格變動是否符合使用者設定條件。
- 透過 Discord Bot 發送通知給使用者。
- 以網站提供輔助性的通知條件設定功能。
