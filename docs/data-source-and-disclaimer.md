# 資料來源與免責聲明

## 資料來源

本專案目前以台灣原價屋（CoolPC）網站上的零售商品價格資訊作為主要追蹤來源。

本專案所提供的價格提醒，是根據系統定期擷取與判斷的結果產生，僅用於協助使用者追蹤特定零件的價格變動。

## 非官方關係聲明

本專案為非官方工具，與原價屋（CoolPC）無任何隸屬、合作、授權、代理或背書關係。

本專案所使用的名稱僅用於描述資料來源，不代表任何商業關係或官方立場。

## 資料限制

本專案僅追蹤特定來源的零售價格，不代表官方建議售價、整體市場行情或所有零售平台的即時價格。

價格提醒結果可能受到以下因素影響：

- 資料來源頁面內容變動
- 商品頁面格式調整
- 抓取週期與通知延遲
- 商品名稱、規格或型號標示差異
- 臨時缺貨、下架或價格異常

因此，本專案不保證資料的即時性、完整性、正確性或持續可用性。

## 使用風險與責任

本專案提供之價格資訊與通知結果，僅作為輔助參考，不應視為購買建議、價格保證或任何形式的交易承諾。

使用者在做出購買決策前，仍應自行以前往原始零售平台確認的價格、庫存、規格與交易條件為準。
