# PCPartAlert

## 專案簡介
透過追蹤台灣原價屋（CoolPC）的電腦零件零售價格，讓使用者能透過 Discord Bot 接收特定商品的價格變動通知，減少手動查價的成本，並更快得知關注商品的價格變化。

## 專案定位
此專案定位為價格提醒工具，主要根據原價屋（CoolPC）的零件零售價格進行追蹤，核心在於追蹤、判斷與通知。  
目前以 Discord Bot 作為主要互動與通知入口；網站僅作為輔助性的細部設定工具，用於讓使用者更快速地設定通知條件，不以完整商品查詢、規格整理或比價瀏覽為主要目標。

## 想解決的問題
1. 手動追蹤價格變動麻煩且耗時。
2. 零件零售價格可能頻繁變動。
3. 使用者不需要為了少數關注商品而反覆手動查詢。
4. 使用者需要能針對特定商品設定追蹤條件並接收通知。

## 使用情境
- 想追蹤某張顯示卡是否低於特定價格。
- 想追蹤某商品是否在短時間內明顯降價。
- 想長期追蹤特定零件，但不想每天手動查價。

## 核心範圍
- 追蹤原價屋（CoolPC）的電腦零件零售價格。
- 提供使用者設定追蹤商品與通知條件的方式。
- 判斷價格變動是否符合使用者設定條件。
- 透過 Discord Bot 發送通知給使用者。
- 以網站提供輔助性的通知條件設定功能。

## 暫不處理
- 完整的商品查詢與瀏覽介面。
- 硬體規格整理與比較功能。
- 組單與估價功能。
- 其他零售平台的價格追蹤。
- 官方網站價格或建議售價追蹤。
- 以網站推播為主的通知系統。
- 市場分析或趨勢研究型內容。
